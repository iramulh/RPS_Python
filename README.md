# RPS_Python
A basic rock-paper-scissors game made in Python that runs in the console.
## How it Works
Uses functions to replicate the following game experience:
```
My wins: 0
Bot wins: 0
Ties: 0
Enter 'rock', 'paper', or 'scissors': rock
You chose rock.
Bot chose scissors.
You win!
My wins: 1
Bot wins: 0
Ties: 0
Enter 'rock', 'paper', or 'scissors': paper
You chose paper.
Bot chose scissors.
Bot win!
My wins: 1
Bot wins: 1
Ties: 0
Enter 'rock', 'paper', or 'scissors': paper
You chose scissors.
Bot chose scissors.
It's a tie!
My wins: 1
Bot wins: 1
Ties: 1
Enter 'rock', 'paper', or 'scissors': quit
Okay, bye.
```
